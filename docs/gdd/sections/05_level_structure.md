# 05 关卡结构（短波次快节奏）

## 波次与显示

- 局内 HUD 显示**当前波次**（如「第 1 波」），波次随时间或条件推进。
- 刷怪难度可随波次提升（生成间隔缩短、炮台/精英比例提高等）。

## 推荐节奏（MVP）

- 每一波持续时间较短（建议约 **10–18 秒**），偏向“快节奏刷波”体验。
- 每一波只刷出**固定数量的敌人**（例如 **6–12 个**，后期波次可略高），强调“清完本波再进入下一波”的节奏，而非持续无限刷怪。
- 随波次推进，整体难度与紧张度逐步上升；前期更教学，后期再引入更高压力组合。
- Boss 战仍控制在 45–75 秒区间（可 1–2 阶段），并放在更靠后的波次作为阶段目标。

## 刷怪与可读性原则

- 同屏高威胁弹源数量受控（建议 1–2 个）。
- 子弹形状/速度/密度保持“新手可躲”。
- 预警优先（激光/冲撞等危险技能先给提示）。

## 敌人刷新与波次升级（新增约定）

- 随着波次提升：
  - 敌人血量与威胁逐波上升（例如 HP 按固定比例随波次递增）。
  - 刷新模式从简单到复杂逐步递进：首波只出基础敌人，不出现会发射子弹的敌人；后续波次再加入炮台与精英。
- **升级触发方式**：
  - **波次结束即升级**：每一波清场后默认弹出一次三选一升级界面，无经验条。
- **Boss 后继续挑战**：见 `05_level_structure_post_boss.md`（威胁等级、敌方倍率、再送一次升级）。
  - 升级完成后再进入下一波，避免在激烈战斗中打断玩家操作。
