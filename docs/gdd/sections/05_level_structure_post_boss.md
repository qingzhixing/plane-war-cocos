# 05b Boss 后继续挑战（续战块 8 波：1～7 小怪 + 第 8 波 Boss）

## 流程

- **首次击破 Boss**（主线第 8 波）：**本局结算** / **继续挑战**。
- **继续挑战**（第一场续战块）：
  - **威胁 +1**，得分乘区 **+8%/层**。
  - **稳态护盾 +1 层**（与词条 `combo_guard` 同一层数逻辑；每 **续关一次** 送一层，可叠）。
  - **连续 3 次三选一升级**（续关奖励），再开 **续战第 1/8 波**（小怪）。
  - **每清一波 → 再三选一升级 → 下一波**（共 **7 波小怪**）；第 7 波升级选完后 **进第 8 波 Boss**（与主线同 Boss 场景，威胁缩放仍生效）。
- **击破续战 Boss 后**：再次 **二选一** — **本局结算** / **接着玩**。
  - **接着玩**：再 **威胁 +1**、**+8% 分**、**护盾 +1 层**、**再 3 次三选一**后进续战第 1/8 波（新一块），可重复。
  - **结算**：进结算页。

## 验收

- 续战 **7 波小怪**每波结束必弹升级；第 7 波升级后 **必出 Boss**；Boss 击破后 **必弹**「结算 / 接着玩」。
- 续战块内 **允许** `_spawn_boss()`（仅第 8 波）；主线仍 **单场首 Boss** 在 wave 8。
