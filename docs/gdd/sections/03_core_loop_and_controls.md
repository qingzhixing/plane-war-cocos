# 03 核心循环与操作（含继续游玩）

## 局外循环（Demo）

1. 进入游戏
2. 开始游戏（MVP：单机体直接进入）
3. 通关/失败结算
4. 返回主菜单再次开局

> Demo 阶段：不做局外永久养成（天赋树/装备/长期数值膨胀）。

## 局内循环（主循环）

- 击败敌人 → 得分/连击
- 波次结束 → 三选一强化（默认弹出）
- 波次推进 → Boss → **结算 或 继续挑战**（继续：威胁+1、**连续 3 次三选一**、**续战 8 波**：7 波小怪 + **第 8 波再打 Boss**，再结算或接着玩）

## 失败与继续游玩（已确认）

- 玩家在局内可以出现“危险状态”或失败，但不鼓励频繁被踢出主循环。
- 建议提供**有限次数的继续机会**（例如每局 1 次），用于降低挫败感。
- 继续后的效果应让玩家有明显感知到“获得了一次补救机会”，但不会长期破坏难度与评分平衡（例如部分恢复生存能力 + 短时间安全窗口）。

## 操作与输入（竖屏、单手 / 键鼠）

- **屏幕方向**：竖屏
- **移动**：
  - **触屏/鼠标**：单指拖拽，手指相对位移 ≈ 飞机相对位移（不做明显缓动，保证灵敏）
  - **键盘**：WASD 或方向键控制移动（与拖拽可同时存在，方便 PC 调试与游玩）
- **射击**：自动射击（固定射速）
- **暂停**：界面“暂停”按钮，或键盘 **P** 键切换暂停（仅在局内、非升级选卡时生效）
- **技能（建议 MVP 就做）**：符卡为 **HUD 右下角短按**；抬起时短按符卡区则 `Main.try_use_spell()`；效果：**周身 360° 符卡弹幕**（`PlayerSpellBullet`）；**不再一键清空全屏敌弹**；符卡弹与 `enemy_bullet` 重叠时仅销毁该敌弹，符卡弹不消失（与弓箭消弹同类）。（代码用 `spell_*`，与副武器炸弹 `bomb_multi` 区分。）
- **一次性升级「自动符卡」（`spell_auto`）**：符卡冷却再 **-50%** + 冷却结束自动 `try_use_spell()`；每局仅 1 次。旧 id `bomb_auto` 仍兼容。
- 飞机位置始终被**限制在屏幕范围内**（预留少量边缘安全距离）
