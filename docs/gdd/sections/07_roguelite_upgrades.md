# 07 Roguelite：升级三选一与强化池（MVP）

## 规则（已确认）

- 击杀敌人可用于统计表现（得分/连击/DPS 等），**不再使用经验值驱动升级**；局内仍可向 `experience_listener` 上报 exp 仅作统计，**不提供「经验增幅」类词条**。
- **升级由波次结束直接触发**：每波清场后默认弹出升级界面，无经验条。
- 每次升级仍为三选一，强化可重复出现并叠加（建议上限 3 级）。

## 强化分类（MVP 目标 12–18 个）

- **主武器（基础子弹）强化词条**：
  - 只影响主武器 `PlayerBullet`，不影响副武器。
  - 示例：
    - **基础火力强化**：主武器伤害 +1（可叠加 3 次）。
    - **高爆弹头**（`damage_percent`）：主武器伤害 **×1.2**（叠乘）；与基础点数、连击加伤一起计入 **HUD 主炮伤害** 显示。
    - **连发扩散**：主武器子弹数量 +1（影响 `_bullet_count`，可叠加 3 次）。
    - **主武器射速**（`fire_rate`）：间隔 **×0.85**；与连击倍率一起算 **理论射速**；**实际 75/s 封顶**；溢出 **每 1% → +0.02 伤**（HUD 合并进 **主炮伤害 … [+…]**，不单独开块）。
    - **专注射击**：减小主武器散射角，子弹速度 +10%。
    - **主武器穿透弹**：主武器子弹获得 1 次穿透。
- **副武器：弓箭（PlayerArrow）词条**：
  - 只影响自动射出的弓箭，不改变主武器行为。
  - 示例：
    - **弓箭装填加速**：弓箭自动射击冷却 -20%。
    - **齐射箭矢**：弓箭齐射数量 +1（影响 `_arrow_shot_count`）；当玩家首次获得该词条时，若弓箭尚未解锁，则同时解锁弓箭并开始以当前数量齐射。
    - **制导优化**：更优先瞄准高威胁目标，箭矢初始速度 +15%。
    - **穿甲箭头**：弓箭获得 1 次穿透。
- **副武器回旋镖（`boomerang_multi`）**：首次解锁；再次选取 **回旋镖齐射 +1**。发射节奏：当前波次全部回收到玩家后再发射下一波。
- **生存**：**稳态护盾**（`combo_guard`）每选一次 **+1 层**；**每次续关**（点「继续挑战」或「接着玩」进入下一块续战）再 **+1 层**。层数可叠加；受击 **消耗 1 层** 免断连；**左侧 HUD「稳态护盾」** 实时显示。
- **符卡**：`spell_cooldown`（旧名 `bomb_cooldown`）缩短冷却；**`spell_auto`（旧 `bomb_auto`）** 一次性自动符卡。
  - **冷却缩短规则**：当符卡最大冷却时间被减少时，**当前剩余冷却时间按同一倍率缩短**（保持进度条百分比不变），并且不会超过新的最大冷却时间，避免出现「最大只需 5 秒、实际还要等 11 秒」的体验落差。
- **副武器炸弹（`bomb_multi` / `bomb_side_cooldown`）**：「挂载炸弹」首次解锁副武器炸弹并齐射 +1；「炸弹装填」缩短炸弹自动发射间隔（未解锁炸弹时不进池）。与玩家/GrazeArea 重叠不引爆、不造成伤害。
- **表现向词条**：连击增益、**评分乘区**（`score_up`）；**已移除**：经验增幅、移动速度增幅、每击固定分、**Boss 特攻 `boss_hunter`**（体感弱）。

## 已移除词条（勿再接入三选一 / 调试表）

| 词条 id | 原因 |
|---------|------|
| `exp_up` | 无经验升级链路，经验仅统计 |
| `move_speed` | 机体速度固定更利于弹幕读招与平衡 |
| `score_flat` | 每击 +固定分体感弱，与 `score_up` 重复维度 |
| `boss_hunter` | 主炮对 Boss +20% 过弱、占卡位 |

## 词条质量约束

- 减少“体感弱、反馈慢”的词条，优先战斗内立即可感知收益。
- 每次三选一中至少保证 1 张为“直接战斗收益词条”（主武器火力、副武器火力或符卡强化），避免选择压力过低。
