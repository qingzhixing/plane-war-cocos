# 01 概览与设计目标（Demo/MVP）

## 项目概览

- **项目代号**：Plane War（暂定）
- **类型**：竖屏 2D 飞行射击 + 轻 Roguelite（局内三选一构筑）
- **风格**：像素风；参考东方符卡氛围，但 **弹幕密度更低、更简单、更休闲**
- **核心操作**：单手拖拽移动 + 自动射击
- **单局时长**：2–5 分钟
- **难度定位**：休闲（容错较高）
- **MVP 规模**：1 架机体 + 波次关卡 + 1 个 Boss + 强化池 + 继续游玩机制
- **发布定位**：Demo / 练习项目

## 设计目标（Design Pillars）

- **轻东方感而不复杂**：符卡命名/阶段提示/演出，弹幕数量少、节奏可读。
- **单手爽快**：移动稳定、自动射击节奏舒适；技能/炸弹用于兜底。
- **短局高反馈**：每 20–40 秒产生一次成长或奖励；2–5 分钟内必见 Boss。
- **系统少但可扩展**：MVP 只做关键系统，后续按“机体/强化/敌人/Boss”横向扩展。
